# 词根字典 PWA

一个智能的英语词根查询工具，支持在线查词、智能学习和收藏功能。

## 🚀 主要功能
- ✅ 查询英语单词的词根和释义
- ✅ 智能在线查词（支持多API）
- ✅ 自动词根分析和学习
- ✅ 用户词根修正和反馈
- ✅ 相似词汇推荐
- ✅ 学习统计和进度跟踪
- ✅ 收藏和导入导出功能
- ✅ 离线使用（PWA）
- ✅ 可添加到iPhone主屏幕

## 🎯 核心特性

### 1. 智能查词系统
- **本地优先**: 首先查询本地词典
- **在线补充**: 本地没有时自动在线查询
- **智能分析**: 即使查询失败也提供词根分析
- **自动学习**: 在线查询结果自动添加到本地词典

### 2. 词根分析引擎
- **前缀识别**: 自动识别常见前缀（un-, re-, pre-等）
- **后缀识别**: 自动识别常见后缀（-tion, -able, -ment等）
- **词根匹配**: 基于大量词根数据库进行匹配
- **智能推断**: 对未知词根进行智能推断

### 3. 学习系统
- **用户修正**: 支持用户修正词根分析结果
- **学习统计**: 跟踪查询历史和学习进度
- **相似推荐**: 基于词根推荐相似词汇
- **数据导出**: 支持学习数据的导入导出

## 📱 使用方法

### 1. 本地测试（Windows）
1. 安装 Python（如果没有）
2. 在项目文件夹打开命令行
3. 运行：`python -m http.server 8000`
4. 浏览器打开：`http://localhost:8000`

### 2. 部署到GitHub Pages（免费）
1. 在GitHub创建新仓库
2. 上传所有文件
3. 在仓库设置中启用GitHub Pages
4. 用iPhone Safari访问生成的网址
5. 点击分享按钮 → "添加到主屏幕"

### 3. 智能学习系统
运行自动学习脚本来扩充词典：
```bash
python auto_learning.py
```

功能包括：
- 批量学习新单词
- 自动词根分析
- 学习统计报告
- 词根模式发现

## 🔧 高级功能

### 词根编辑
- 点击词根旁的编辑按钮可以修正词根信息
- 修正结果会用于改进未来的分析
- 支持用户贡献词根知识

### 学习统计
- 查看总查询次数和学习进度
- 分析最常用的词根
- 导出学习数据进行备份

### 相似词汇
- 基于词根相似性推荐相关词汇
- 帮助扩展词汇量
- 发现词汇之间的联系

## 📊 数据管理

### 词典数据结构
```javascript
{
  "单词": {
    "word": "单词",
    "gloss": "释义",
    "phonetic": "音标",
    "roots": [
      { "root": "词根1", "meaning": "含义1" },
      { "root": "词根2", "meaning": "含义2" }
    ],
    "source": "数据来源"
  }
}
```

### 添加词根数据
1. **手动添加**: 编辑 `dictionary.json` 文件
2. **自动学习**: 使用 `auto_learning.py` 脚本
3. **用户修正**: 通过界面修正词根信息

## 🛠️ 技术架构

## 📝 注意事项
- 数据存储在浏览器本地，清除浏览器数据会丢失收藏
- 需要HTTPS或localhost才能使用PWA功能
- GitHub Pages自动提供HTTPS
- 在线查询需要网络连接
- API查询有频率限制，建议适度使用
